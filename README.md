# Tendencias de Mercado 2025-2026 — Gamonal

Reporte interactivo de tendencias de mercado para marketing y comunicaciones B2B, con foco en el comprador actual y el mercado mexicano.

## Estructura del estudio

5 capítulos con hallazgos verificados en fuente primaria (fuentes 2022 o posteriores).

- **01 — El comprador B2B**: cómo investiga, qué le importa, qué le hace cerrar.
- **02 — Atención y canales**: dónde está la atención, IA, LinkedIn, sharing.
- **03 — Producto y experiencia**: lo que tu sitio, velocidad y producto deben entregar.
- **04 — México**: el mercado, el comprador local, banca y fintech.
- **05 — Marca y presupuesto**: por qué la marca convierte y cómo asignar el mix.

## Modo edición

El sitio incluye un modo edición protegido, activable con `Ctrl + Shift + E` (pide contraseña; se define en la constante `MASTER_PW` dentro de `index.html`).

Una vez activado:

- Click "Editar" en la toolbar.
- Cambios autoguardados en el navegador (localStorage, claves `tendencias-mercado-*`).
- Botón "Descargar versión pública" exporta `tendencias_mercado.html` con los cambios incrustados.
- Publicación directa a GitHub disponible con un token personal (repositorio configurado: `andresgamonalm/tendencias-mercado`, rama `main`).

## Cómo desplegarlo

### Cloudflare Pages (recomendado)

1. `dash.cloudflare.com → Workers & Pages → Pages → Connect to Git`
2. Selecciona este repositorio.
3. Branch: `main`. Build command y output: vacíos.
4. Deploy.

### GitHub Pages

1. Settings → Pages → Source: `main` branch, root.
